Pièges prioritaires — Maths Spé

  • PROBA : "Au moins un" = complément 1 − P(aucun). Ne jamais calculer directement.
  • BIENAYMÉ-TCHEBYCHEV : L'inégalité majore P(|Y−E|≥a). Pour minorer P(|Y−E|<a), passer au complément.
  • V(Fn) : p(1−p)/n — diviser par n, PAS par n².
  • SUITES : Hérédité sur encadrement = appliquer h aux TROIS membres. Prouver la convergence AVANT d'écrire l = h(l).
  • ÉQ. DIFF. : y' + ay = 0 donne e^(−ax), pas e^(+ax). Toujours fixer C avec la condition initiale.
  • GÉOMÉTRIE : Distance point-plan : valeur absolue au numérateur. Projeté orthogonal : direction = vecteur normal uniquement.
  • IPP avec ln : Dériver le ln, intégrer le polynôme. Ne pas oublier le crochet.
  • CONVEXITÉ : f convexe ⟺ f'' ≥ 0 (≠ signe de f'). L'inégalité tangente découle de la concavité sur l'intervalle.
