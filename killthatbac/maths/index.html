P1
PROBA — "Au moins un" = 1 − P(aucun)
Ne JAMAIS calculer P(X≥1) directement. Toujours passer par le complément : P(au moins un succès) = 1 − P(X=0) = 1 − (1−p)^n
▼
ℹ Méthode — Loi binomiale (fiche)
Justifier les 3 conditions : tirage avec remise, épreuves indépendantes, deux issues. Donner n et p.
À la calculatrice : P(X=k) et P(X≤k).
Au moins un succès = 1 − P(X=0) = 1 − (1−p)^n.
E(X) = np  ·  V(X) = np(1−p).
PRIORITÉ 1 AN 2026 J1 — Ex.1 Partie B Plateforme streaming
X ~ B(8, 0,4125). La question demande : sur n semaines consécutives, quelle est la probabilité qu'au moins une semaine ait eu strictement plus de 4 abonnés ayant activé H ?
→ q_n = 1 − P(X=0 chaque semaine) = 1 − (0,5875)^n
Puis trouver le plus petit n tel que q_n ≥ 0,999 → résoudre par ln : n ≥ ln(0,001)/ln(0,5875).
Piège : calculer P(X≥1) directement avec la loi binomiale sur n semaines — impossible. Il faut modéliser la réussite d'une semaine, puis enchaîner sur n semaines indépendantes.
Sujet Corrigé
Prioritaire AN 2025 J1 — Ex.1 Partie B.2 Serveurs informatiques
P(serveur S fonctionne) = 0,855. Sur n serveurs en parallèle : probabilité qu'au moins un fonctionne = p_n = 1 − (1−0,855)^n = 1 − (0,145)^n.
Trouver n tel que p_n ≥ 0,99 → n ≥ ln(0,01)/ln(0,145) → n = 2.
Piège : confondre "la proba qu'au moins un fonctionne" avec "la proba que le premier fonctionne". La structure est indépendante : 1 − P(tous échouent).

P2
PROBA — Bienaymé-Tchebychev : sens de l'inégalité
BT donne : P(|Y − E(Y)| ≥ a) ≤ V(Y)/a² (majoration).
Pour minorer P(|Y − E(Y)| < a) : passer au complément → P(...<a) ≥ 1 − V(Y)/a².
Présent dans les 6 sujets prioritaires.
▼
ℹ Méthode — Bienaymé-Tchebychev (fiche)
P(|Y − E(Y)| ≥ a) ≤ V(Y) / a² → majoration de la proba de grand écart.
Pour minorer P(|Y−E| < a) : passer au complément → ≥ 1 − V(Y)/a².
V(Fn) = p(1−p)/n (fréquence empirique).  V(Mn) = σ²/n (moyenne empirique).
Test de conformité : comparer l'écart observé à la borne — ne jamais conclure "c'est certain".
PRIORITÉ 1 AN 2026 J2 — Ex.1 Partie B Tomates commercialisables
F_n = fréquence de tomates non commercialisables. p = 0,09, V(F_n) = p(1−p)/n = 0,0819/n.
BT : P(|F_n − 0,09| ≥ 0,05) ≤ 0,0819/(n × 0,0025) = 32,76/n.
Donc P(0,04 < F_n < 0,14) ≥ 1 − 32,76/n.
Test : n=500, F_obs = 55/500 = 0,11. Est-ce que 0,11 ∈ ]0,04 ; 0,14[ ? Oui → conforme à p=9%.
Piège 1 : conclure "la machine est bien réglée" → non, c'est une probabilité. On dit seulement que le résultat est compatible avec p=0,09.
Piège 2 : V(F_n) = p(1−p)/n, pas p(1−p)/n².
Sujet Corrigé
Prioritaire AN 2025 J2 — Ex.1 Partie B Basketball

P3
SUITES — Hérédité sur encadrement : appliquer h aux 3 membres
Si h est croissante et a ≤ u_n ≤ b, alors h(a) ≤ h(u_n) ≤ h(b).
Ne pas oublier de montrer séparément que u_{n+1} ≤ u_n (ou u_n ≤ u_{n+1}) selon le sens de la monotonie.
▼
ℹ Méthode — Suites : encadrement et convergence (fiche)
Si h croissante et a ≤ u_n ≤ b → appliquer h aux 3 membres : h(a) ≤ h(u_n) ≤ h(b).
Suite décroissante minorée (ou croissante majorée) → convergente (théorème suite monotone bornée).
Limite l : justifier la convergence D'ABORD, puis résoudre l = h(l). Choisir la racine cohérente avec l'encadrement.
PRIORITÉ 1 AN 2026 J1 — Ex.2 Partie A Posidonie marine
u_{n+1} = 4 − 4/u_n, h(x) = 4 − 4/x croissante sur ]0;+∞[.
Montrer : 2 ≤ u_{n+1} ≤ u_n ≤ 4.
Étape 1 — appliquer h à l'encadrement 2 ≤ u_n ≤ 4 : h(2)=2 ≤ h(u_n) ≤ h(4)=3 ≤ 4. Donc 2 ≤ u_{n+1} ≤ 4. ✓
Étape 2 — montrer u_{n+1} ≤ u_n : 4−4/u_n ≤ u_n ⟺ u_n²−4u_n+4 ≥ 0 ⟺ (u_n−2)² ≥ 0. ✓
Conclusion : suite décroissante et minorée par 2 → converge. Limite l : l = 4−4/l → l²−4l+4=0 → l=2.
Piège classique : ne faire que l'encadrement h(a)≤h(u_n)≤h(b) sans montrer u_{n+1}≤u_n séparément. Les deux inégalités sont demandées.
Sujet Corrigé
PRIORITÉ 1 AN 2026 J2 — Ex.2 Partie B Suite f(x)=2x/√(1+x²)
u_{n+1} = f(u_n) = 2u_n/√(1+u_n²), f croissante. Montrer : 1 ≤ u_n ≤ u_{n+1} < √3.
Encadrement croissant (u_{n+1} ≥ u_n) : f(x) ≥ x sur [1;√3[ car 2x/√(1+x²) ≥ x ⟺ 4x² ≥ x²(1+x²) ⟺ 3 ≥ x².
Borne sup : f croissante et f(√3) = 2√3/2 = √3, donc u_{n+1} = f(u_n) < f(√3) = √3 si u_n < √3.
Piège : sens de l'inégalité inversé sur un des côtés. Ici la suite est CROISSANTE et majorée — différent de P3 ex.1 où elle est décroissante.
Sujet Corrigé

P4
SUITES — Suite géométrique auxiliaire : vérifier la raison pour tout n
Trouver v_n = φ(u_n) tel que v_{n+1} = q × v_n pour TOUT n (pas seulement un exemple).
Donner v_0, écrire v_n = v_0 × q^n, puis exprimer u_n en fonction de v_n.
▼
ℹ Méthode — Suite géométrique auxiliaire (fiche)
Poser v_n = φ(u_n). Montrer v_{n+1} = q × v_n pour TOUT n (par calcul algébrique, pas sur un exemple).
Donner v_0. Écrire v_n = v_0 × q^n. Revenir à u_n par la relation inverse.
Python : boucle while pour un seuil (compteur n++), boucle for range(p) avec L.append(S).
PRIORITÉ 1 AN 2026 J2 — Ex.2 Partie B Suite f(x)=2x/√(1+x²)
Poser v_n = u_n² / (3 − u_n²).
Calcul : v_{n+1} = u_{n+1}²/(3−u_{n+1}²) = [4u_n²/(1+u_n²)] / [3 − 4u_n²/(1+u_n²)]
= 4u_n² / (3+3u_n²−4u_n²) = 4u_n² / (3−u_n²) = 4 × v_n.
Raison q=4. v_0 = 1²/(3−1) = 1/2. Donc v_n = (1/2)×4^n.
Retour à u_n : u_n² = 3v_n/(1+v_n) → u_n = √(3×(1/2)×4^n / (1 + (1/2)×4^n)) → lim = √3.
Piège : vérifier que v_{n+1}=4v_n en substituant U_{n+1}=f(u_n) explicitement — ne pas juste dire "on pose v_n" sans montrer la relation de récurrence.
Sujet Corrigé
Prioritaire AN 2025 J2 — Ex.2 Méthode de Briggs (approx. ln 2)
u_0=2, u_{n+1}=√u_n. Poser v_n = ln(u_n).
v_{n+1} = ln(√u_n) = (1/2)ln(u_n) = (1/2)×v_n. Raison q=1/2.
v_0 = ln(2). Donc v_n = ln(2)/2^n.
Clé : ln(2) = 2^n × v_n = 2^n × ln(u_n). Pour grand n, u_n ≈ 1 donc ln(u_n) ≈ u_n − 1, d'où ln(2) ≈ 2^n × (u_n − 1).
Piège : ln(u_{n+1}) = ln(√u_n) = (1/2)ln(u_n) est direct, mais certains oublient que √u_n = u_n^(1/2) et utilisent ln(u_n/2) — erreur.

P5
ÉQ. DIFF. — Signe dans l'exponentielle
y' + ay = 0 → solution générale : y = C × e^(−ax).
Le signe est TOUJOURS négatif dans l'exponentielle. Toujours fixer C avec la condition initiale.
▼
ℹ Méthode — Équation différentielle y' + ay = b (fiche)
Homogène y' + ay = 0 → y = C × e^(−ax) (signe toujours négatif).
Particulière constante : y_p = b/a. Si y_p est fournie : juste vérifier en dérivant et substituant.
Solution générale = y_p + y_h. Fixer C avec la condition initiale : y(x₀) = y₀.
PRIORITÉ 1 AN 2026 J1 — Ex.2 Partie B Posidonie — modèle continu
Équation : y' + y = 2 (a=1, b=2).
Homogène y'+y=0 → y_h = Ce^(−t) (signe NÉGATIF).
Particulière constante : y_p = 2 (car y_p' = 0, 0+2=2 ✓).
Solution générale : p(t) = Ce^(−t) + 2.
Condition initiale p(0)=4 → C+2=4 → C=2.
p(t) = 2e^(−t) + 2. Limite en +∞ : 2 (population d'équilibre).
Piège le plus courant : écrire Ce^(+t). La solution de y'+y=0 est bien e^(−t) car (e^(−t))' = −e^(−t) et −e^(−t)+e^(−t)=0.
Sujet Corrigé

P6
GÉOMÉTRIE — Distance point-plan : ne pas oublier la valeur absolue
d(M, plan ax+by+cz+d=0) = |ax₀+by₀+cz₀+d| / √(a²+b²+c²).
La valeur absolue au numérateur est obligatoire — le résultat est toujours positif.
▼
ℹ Méthode — Géométrie : distance point-plan (fiche)
Plan ax+by+cz+d=0, point M(x₀,y₀,z₀) :
d(M, plan) = |ax₀+by₀+cz₀+d| / √(a²+b²+c²) — valeur absolue obligatoire.
Volume tétraèdre = (1/3) × Aire(base) × hauteur.
Double méthode : calculer V d'abord → d = 3V / Aire(base).
PRIORITÉ 1 AN 2026 J1 — Ex.3 Partie B Pyramide SABCD
Plan (SBC) : équation 2y + z − 2 = 0. Point O(0,0,0).
d(O, plan) = |2×0 + 0 − 2| / √(0²+2²+1²) = 2/√5 = 2√5/5.
Vérification : volume pyramide O-SBC = (1/3) × aire(SBC) × hauteur.
Aire(SBC) = √5 → hauteur = 3×V / Aire(SBC).
Piège : oublier les barres de valeur absolue et obtenir un résultat négatif → distance negative = erreur garantie. Rationaliser la fraction (2/√5 → 2√5/5).
Sujet Corrigé
Prioritaire AN 2025 J2 — Ex.2 Partie B Tétraèdre ABCS
Plan (ABC) : équation x + 2y + 4z − 7 = 0. Point S(−1/2, 1, 4).
d(S, plan) = |−1/2 + 2×1 + 4×4 − 7| / √(1+4+16) = |−1/2+2+16−7| / √21 = |10,5| / √21 = 10,5/√21 = √21/2.
Piège : −1/2 + 2 + 16 − 7 = 10,5 (attention au demi-entier). 10,5/√21 = 10,5×√21/21 = (1/2)×√21 = √21/2.

P7
GÉOMÉTRIE — Projeté orthogonal : direction = vecteur normal
Projeté H de M sur plan P :
1) écrire la droite (MH) : M + t×n où n = vecteur normal du plan.
2) substituer dans l'équation du plan → trouver t → calculer H.
▼
ℹ Méthode — Projeté orthogonal (fiche)
H = projeté de M sur plan P :
1. Droite (MH) : M + t × n où n = vecteur normal du plan (pas un vecteur quelconque).
2. Substituer dans l'équation de P → trouver t → calculer H.
Vérif : MH · v = 0 pour tout vecteur v du plan (produit scalaire nul).
Prioritaire Métropole 2025 J2 — Ex.2 Tétraèdre ABCS
Plan (ABC) : x + 2y + 4z − 7 = 0. Vecteur normal n = (1, 2, 4). Point S(−1/2, 1, 4).
Droite (SH) : (−1/2+t ; 1+2t ; 4+4t).
Substituer : (−1/2+t) + 2(1+2t) + 4(4+4t) − 7 = 0
→ −1/2 + t + 2 + 4t + 16 + 16t − 7 = 0 → 21t + 10,5 = 0 → t = −1/2.
H = (−1/2−1/2 ; 1−1 ; 4−2) = (−1 ; 0 ; 2).
Piège : prendre n'importe quel vecteur du plan comme direction (ex : AB ou AC) — le projeté serait faux. La direction DOIT être le vecteur normal.

P8
ANALYSE — Intégration par parties double avec ln
Pour ∫x(ln x)² dx : dériver (ln x)², intégrer x.
1ère IPP → apparition de ∫x ln x dx → 2ème IPP nécessaire. Ne pas oublier le crochet [ ]ₐᵇ.
▼
ℹ Méthode — Intégration par parties (fiche)
∫u·v' = [u·v]ₐᵇ − ∫u'·v. Règle : dériver le ln, intégrer le polynôme.
∫x ln x : u=ln x, v'=x → une IPP.
∫x(ln x)² : u=(ln x)², v'=x → première IPP → apparition de ∫x ln x → deuxième IPP.
Ne pas oublier d'évaluer le crochet [...]ₐᵇ aux bornes après chaque IPP.
PRIORITÉ 1 AN 2026 J2 — Ex.4 et AN 2025 J2 — Ex.4 f(x) = x(ln x)²
IPP 1 : u=(ln x)², v'=x → u'=2ln(x)/x, v=x²/2.
∫x(ln x)²dx = [x²(ln x)²/2] − ∫x ln x dx.

IPP 2 : u=ln x, v'=x → u'=1/x, v=x²/2.
∫x ln x dx = [x² ln x/2] − ∫x/2 dx = x² ln x/2 − x²/4 + C.

Résultat final : ∫x(ln x)²dx = x²(ln x)²/2 − x² ln x/2 + x²/4 + C.

Sur [α,1] (α→0) : l'intégrale vaut 1/4 car lim x²(ln x)²=0 et lim x² ln x=0 en 0.
Piège 1 : intervertir u et v' → dériver x et intégrer (ln x)². Impossible car ∫(ln x)² dx n'est pas immédiat.
Piège 2 : oublier d'évaluer le crochet aux bornes après chaque IPP.

P9
ANALYSE — Convexité → inégalité tangente
f convexe (f''≥0) sur I → courbe au-dessus de toute tangente.
f concave (f''≤0) sur I → courbe en-dessous de toute tangente.
Ne pas confondre convexité (signe f'') avec monotonie (signe f').
▼
ℹ Méthode — Convexité et inégalité tangente (fiche)
f'' ≥ 0 → convexe → courbe au-dessus de toute tangente → f(x) ≥ T(x).
f'' ≤ 0 → concave → courbe en-dessous de toute tangente → f(x) ≤ T(x).
Point d'inflexion : f'' change de signe (≠ extremum = f' change de signe).
Tangente en a : y = f'(a)(x−a) + f(a). Justifier l'intervalle sur lequel la convexité est établie.
PRIORITÉ 1 AN 2026 J1 — Ex.4 f(x) = 5ln(x²+1) − 3x
f''(x) = (−10x²+10)/(x²+1)² fourni. Pour x ≥ 1 : f''(x) ≤ 0 → f concave sur [1;+∞[.
Tangente en x=1 : f(1)=5ln(2)−3, f'(1)=7.
Tangente : T(x) = 7(x−1) + 5ln(2)−3 = 7x + 5ln(2) − 10.
Concavité → courbe EN DESSOUS de T → f(x) ≤ T(x) pour x ≥ 1.
→ 5ln(x²+1)−3x ≤ 7x+5ln(2)−10 → ln(x²+1) ≤ x + ln(2) − 1.
Piège : si concave → courbe sous la tangente (inégalité ≤). Si convexe → courbe sur la tangente (inégalité ≥). Facile à inverser sous le stress.
Sujet Corrigé
Prioritaire Métropole 2025 J1 — Ex.2 f(x) = x(2(ln x)²−3 ln x+2)
f''(x) = (1/x)(4 ln x + 1). f''(x)=0 ⟺ ln x = −1/4 ⟺ x = e^(−1/4).
Point d'inflexion en x₀ = e^(−1/4).
Pour x < e^(−1/4) : f''<0 → concave. Pour x > e^(−1/4) : f''>0 → convexe.
Aire entre Cf et la tangente au point B : utiliser la concavité/convexité pour signer l'intégrale.
Piège : confondre point d'inflexion (f'' change de signe) et extremum (f' change de signe).
